# ü©∫ ASD Behavioral Screening Assistant

An end-to-end Machine Learning web application. This tool assists in the preliminary screening of Autism Spectrum Disorder (ASD) traits by analyzing clinical AQ-10 parameters through a Random Forest Classifier.

---

## üöÄ Key Features
- **Real-Time AI Prediction:** Instant classification of behavioral traits based on user input.
- **Dynamic Diagnostic Dashboard:** Generates a 4-panel analysis including:
  - **Age Density Plot:** Situates the patient within clinical dataset norms.
  - **Response Intensity Heatmap:** Visualizes the strength of specific behavioral markers.
  - **Trait Cluster Analysis:** Groups AQ-10 questions into clinical domains (Social, Communication, etc.).
  - **Feature Importance:** Displays the AI's primary decision-making factors.

## Installation & Usage
Clone the repository:

```
git clone https://github.com/aishu18-github/Autism-Predictor.git
cd Autism-Predictor
```
Install dependencies:

```
pip install -r requirements.txt
```
Run the application:

```
python app.py
```
The app will be available at http://127.0.0.1:5000/
